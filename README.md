# AIzeroMD - WhatsApp Bot

A powerful, elegant, and modular WhatsApp bot built with 💖 by WBK.

> ⚠️ **WARNING:** SOURCE CODE INI TIDAK AKAN DIUPDATE KALAU BELUM ADA REPORT ERROR

## 🚀 Getting Started

```bash
git clone https://github.com/wbagazk/AIzeroMD.git
cd AIzeroMD
npm install
npm start
```

## 📦 Requirements

- Requires username, password, and IP address to connect to this Bot
- Internet connection

## 🔐 Security Tips

```diff
- Never share your session file, username, password, and IP address!
+ Use environment variables to store secrets!
```
